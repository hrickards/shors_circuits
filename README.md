Shor's Circuits
===============
Symbolic simulation of quantum circuits.

To try out, run `padrino start` and go to `localhost:3000/circuits`.

Requirements
------------
* Python 2.7 with sympy
* pdf2vg
* pdfcrop (`texlive-extra-utils`)
* pdflatex
* inkscape (SVG to PNG)
* Ruby with gems in Gemfile
